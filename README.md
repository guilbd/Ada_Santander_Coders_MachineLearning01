# Projeto de Regressão com Machine Learning

## Descrição do Projeto

Este projeto visa aplicar técnicas de regressão em um conjunto de dados específico. O projeto é composto por análises exploratórias, pré-processamento de dados e modelagem de Machine Learning.

## Estrutura do Projeto

O projeto está estruturado em um Jupyter Notebook e contém as seguintes seções:

1. Introdução
2. Importação de Bibliotecas
3. Carregamento de Dados
4. Análise Exploratória de Dados (EDA)
5. Pré-processamento de Dados
6. Modelagem de Machine Learning
7. Avaliação do Modelo
8. Conclusão

## Pré-requisitos

- Python 3.x
- Jupyter Notebook
- Bibliotecas: pandas, numpy, scikit-learn, matplotlib, seaborn

## Como Executar

1. Clone este repositório.
2. Instale as dependências usando pip:

    ```bash
    pip install -r requirements.txt
    ```

3. Abra o Jupyter Notebook:

    ```bash
    jupyter notebook
    ```

4. Navegue até o diretório do projeto e abra o notebook `Projeto_Regressao_ML1.ipynb`.
5. Execute todas as células do notebook.
